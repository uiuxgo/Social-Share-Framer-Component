# Social Share Framer Component

Цей репозиторій містить готовий до використання кастомний компонент (`SocialShareFramerComponent.tsx`) для [Framer](https://www.framer.com/), який дозволяє легко додавати красиві та функціональні кнопки поширення у соціальних мережах на ваші вебсайти.

## 🚀 Основні можливості

- **Підтримка багатьох платформ:** Facebook, Twitter / X, LinkedIn, Reddit, Pinterest, WhatsApp, Telegram, Email, а також кнопка "Копіювати посилання" (Copy URL) у буфер обміну.
- **Два режими посилання (Share Link):**
  - **Current Page:** Автоматично бере URL поточної сторінки сайту.
  - **Custom URL:** Дозволяє вручну вказати будь-яке посилання.
- **Гнучка кастомізація зовнішнього вигляду:**
  - Налаштування кольорів іконки та фону (використання офіційних брендових кольорів кожної платформи або власних кольорів).
  - Налаштування ефектів при наведенні (Hover Effects) для фону, обводки та іконок.
  - Гнучкі налаштування відступів (`padding`) та скруглення кутів (`border-radius`) як для всіх сторін разом, так і для кожної окремо (Top, Right, Bottom, Left).
  - Налаштування розміру іконки.
  - Можливість додавати обводку (`border`) зі своїми стилями (колір, товщина, тип лінії).
- **Власні іконки:** Замість стандартних векторних іконок ви можете завантажити власну SVG-іконку (через код) або растрове зображення.
- **Префікси та Суфікси:** Спеціальне налаштування для кнопки "Копіювати посилання", щоб додавати текст перед та після посилання (наприклад: "Подивіться на це: [Тут буде URL] #Awesome").
- **Відкриття у новій вкладці:** Вбудована опція для відкриття посилань без закриття вашого сайту.

## 🛠 Як використовувати у Framer

1. Відкрийте ваш проєкт у **Framer**.
2. Відкрийте панель **Assets** (зліва) та перейдіть на вкладку **Code**.
3. Натисніть кнопку плюса (`+`) та створіть новий компонент з коду (наприклад, назвіть його `SocialShare`).
4. Скопіюйте весь код з файлу `SocialShareFramerComponent.tsx` і вставте його у щойно створений файл компонента у Framer, замінивши стандартний (дефолтний) код.
5. Збережіть файл (`Cmd + S` або `Ctrl + S`).
6. Відкрийте сторінку (Canvas), куди ви хочете додати кнопку. Перетягніть компонент `Social Share` з панелі **Assets -> Components** на полотно.
7. Використовуйте **Панель властивостей (Properties panel)** справа, щоб налаштувати платформу, кольори, розміри та інші параметри компонента на ваш смак.
